### Introduction to Java

#### 1. What is a Program

A program is a series of **ordered instructions** written for a computer to perform certain operations or solve a problem.

My personal understanding: A methodical solution crafted according to logical steps.

#### 2. The Birth of Java

1. In 1990, Sun Microsystems launched the Green Project.
2. In 1992, the Oak language was created, which later became Java.
3. In 1994, James Gosling presented Java at a Silicon Valley conference, astonishing the world.
4. In 1995, Sun Microsystems officially released the first version of Java.
5. In 2009, Oracle Corporation acquired Sun Microsystems and released Java 7 in 2011.

#### 3. Java Classifications

1. **Java SE (Java Standard Edition)**
   - Provides a complete API, geared towards desktop applications.
2. **Java EE (Java Enterprise Edition)**
   - Designed for developing enterprise-level applications.
3. **Java ME (Java Micro Edition)**
   - Rarely used now, it was very popular during the Nokia era.

#### 4. Features of Java (Understandable through extensive coding)

1. Java is an object-oriented language.
2. Java is robust, with strong type checking, exception handling, and automatic garbage collection ensuring program robustness.
3. Java is platform-independent.
4. Java is an interpreted language.
   - Compiled languages produce code that the machine can directly execute.
   - Interpreted languages produce code that requires an interpreter to execute.

#### 5. Java Runtime Mechanism

Write a `Test.java` file, compile it into a `Test.class` file, and run it through the JVM (Java Virtual Machine).

(In simple terms: Write a human-readable file, compile it with `javac` into a class file, and run the class file using the JVM. The JVM acts as an intermediary to handle differences between systems, enabling the code to run on any machine with a JVM installed.)

1. **JDK (Java Development Kit):** A development toolkit that includes the JRE and tools like `java`, `javac`, `javadoc`, `javap`, etc.
2. **JRE (Java Runtime Environment):** Includes the JVM and core Java classes.
3. **JVM (Java Virtual Machine):** Runs the code, facilitating communication between the device and the code.
